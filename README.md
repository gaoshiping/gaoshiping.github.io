# Shiping Gao

This repository hosts the personal academic website for Shiping Gao.

The site is built with Jekyll and the Academic Pages template, and is configured for GitHub Pages at:

https://gaoshiping.github.io

## Site Content

- Home: research profile and current academic status
- Publications: conference papers, submissions, and journal articles
- Research: selected research projects in NLP, LLM alignment, reward modeling, and reasoning
- Teaching: teaching assistant experience
- CV: education, publications, research experience, teaching, honors, and technical skills

## Local Preview

Install Ruby dependencies, then run:

```bash
bundle exec jekyll serve -l -H localhost
```

The local site will be available at `http://localhost:4000`.

## Maintenance Notes

- Site-wide metadata lives in `_config.yml`.
- Top navigation lives in `_data/navigation.yml`.
- Main pages live in `_pages/`.
- Publication entries live in `_publications/`.
- Teaching entries live in `_teaching/`.
